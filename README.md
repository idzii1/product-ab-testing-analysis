# A/B Testing & Product Analytics

## Project Overview
This project simulates an A/B test to compare user behavior between two product variants.

## Tools
- Python (pandas, matplotlib)
- Basic statistics

## Analysis
- Conversion rate calculation
- Performance comparison of variants
- Data visualization

## Key Insights
- Variant B has a higher conversion rate than Variant A
- Data shows a clear difference in user behavior

## Business Recommendation
Implement Variant B to improve conversion rate and increase product performance.

## Conclusion
This project demonstrates how data-driven experiments can support product decisions.
